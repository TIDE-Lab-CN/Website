# TIDE-Lab Website

TIDE-Lab 的网站。

## 内容结构

`src/content` 目录为网站内容目录，所有的内容都在其中。

- `content/activities/news`: 此目录中每一个文件夹作为一篇风采新闻，每个文件夹中都应有一个 `index.md` 作为主文件。
- `content/activities/photots`: 此目录中所有图片都将作为风采图片展示，图片名即为图片描述。
- `content/members/{faculty|student|alumni}`: 此三个目录分别对应教师，学生，校友，每个文件夹中都应有一个 `index.md` 作为主文件。
